# Demo V1.7

# Feedback Collection

## Purpose

เพิ่มช่องทางรับความคิดเห็นจาก

- นักเรียน
    
- ผู้ปกครอง
    
- ครู
    
- ผู้ทดลองใช้งาน
    

เพื่อใช้ในการพัฒนา MVP ต่อไป

---

# Why

ปัจจุบัน Demo สามารถแสดง

- Learning Flow
    
- Student Note
    
- Parent Summary
    
- Real-world Learning
    
- Learning Reflection
    

ได้แล้ว

สิ่งที่สำคัญที่สุดในขั้นถัดไปคือ

```text
User Feedback
```

ไม่ใช่การเพิ่ม Feature ใหม่

---

# Goal

หลังจากผู้ใช้ทดลอง Demo

สามารถส่งความคิดเห็นกลับมาได้ง่าย

ภายใน 1 คลิก

---

# UI Design

## Feedback Card

ตำแหน่ง

- ด้านล่างของหน้า Demo
    
- หรือด้านขวาใต้ Parent Summary
    

---

## Example

🙏 ช่วยพัฒนา AI Tutor คณิตศาสตร์ ม.2

โครงการนี้เริ่มต้นจากความตั้งใจของพ่อคนหนึ่ง  
ที่อยากช่วยให้ลูกเรียนคณิตศาสตร์ได้ดีขึ้น

ความคิดเห็นของคุณจะช่วยให้เราพัฒนาระบบให้ตอบโจทย์นักเรียนและผู้ปกครองได้ดียิ่งขึ้น

[ ส่งความคิดเห็น ]
