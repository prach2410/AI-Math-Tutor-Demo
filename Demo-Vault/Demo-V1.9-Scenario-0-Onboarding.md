# Demo V1.9

# Scenario 0 - ทดลองใช้ AI Tutor

## Purpose

ช่วยให้นักเรียนที่เข้ามาใช้งานครั้งแรก

- รู้วิธีใช้ระบบ
    
- รู้ว่าต้องกดอะไร
    
- รู้ว่าตอบผิดได้
    
- รู้ว่าขอความช่วยเหลือได้
    

ก่อนเข้าสู่บทเรียนจริง

---

# Background

จาก User Testing

พบว่า

นักเรียนบางคน

```text
งง
ไม่รู้จะเริ่มอย่างไร
ไม่รู้ว่าต้องกดอะไร
```

ปัญหาไม่ได้อยู่ที่เนื้อหาคณิตศาสตร์

แต่เป็น

```text
First-Time User Experience
```

ก่อนเริ่มเรียนจริง

ไม่ต้องกลัวตอบผิดครับ

## AI Tutor

# Step 2

# ลองใช้คำใบ้

## AI Tutor

ลองกดปุ่ม

💡 คำใบ้

## Student Action

แต่จะช่วยให้หนูคิดต่อได้

# Step 3

# ลองใช้ "ช่วยเริ่มให้หน่อย"

## AI Tutor

ต่อไปลองกด

🆘 ช่วยเริ่มให้หน่อย

ดูนะครับ

---

## Student Action

---

# Step 4

💡 ใช้ในชีวิตจริง

👨‍👩‍👧 สรุปสำหรับผู้ปกครอง

# พร้อมเรียนจริง

# Product Insight

Students were confused not because mathematics was difficult.

Students were confused because they did not know how to use the system.

The problem was onboarding, not learning.
